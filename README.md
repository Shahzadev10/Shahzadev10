# 💫 About Me:
Artificial Intelligence and Generative AI. What started as curiosity about how intelligent systems work has turned into hands-on experience building applications — and learning how to actually ship and run them in the real world.<br><br>So far, I've worked with Python, NumPy, Pandas, Matplotlib, Tkinter, and Pygame. I'm especially interested in Generative AI, and I've used the Gemini API in my own projects to build apps that can adapt and generate content, not just follow fixed rules.<br><br>I'm currently expanding into Machine Learning, Deep Learning, and MLOps — learning tools like FastAPI, Docker, and MLflow to understand how AI models are trained, tracked, and deployed as real, reliable systems, not just notebooks.
